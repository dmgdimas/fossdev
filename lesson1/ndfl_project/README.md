# Публикация NDFL-калькулятора в TestPyPI

## Введение
В рамках данного задания проект по расчету НДФЛ (налога на доходы физических лиц) был преобразован из локального скрипта в распространяемый Python-пакет. Основная цель — обеспечить воспроизводимость сборки и удобство установки через стандартный менеджер пакетов `pip`.

**Репозиторий проекта:** [https://github.com/dmgdimas/fossdev](https://github.com/dmgdimas/fossdev)

## Техническая реализация

### 1. Структура пакета (Src Layout)
Для исключения конфликтов имен при импорте была выбрана структура `src-layout`. Весь исходный код перемещен в папку `src/ndfl_calc_dmg/`. Это гарантирует, что тесты запускаются против установленного пакета, а не против локальных файлов.

### 2. Конфигурация сборки (pyproject.toml)
Использован современный стандарт `pyproject.toml`. В нем описаны:
- Метаданные (авторство, описание, версия).
- Ссылки на репозиторий (GitHub).
- Зависимости (библиотека `requests`).
- Настройки сборщика `setuptools`.

### 3. Автоматизация жизненного цикла (Makefile)
Управление проектом осуществляется через `Makefile`. Реализован **Stateless-подход**: мы не активируем виртуальное окружение вручную через `source activate`. Вместо этого команды обращаются напрямую к интерпретатору внутри `.venv/bin/`.

**Ключевые команды:**
- `make install` — Подготовка изолированного окружения.
- `make check` — Запуск проверок качества (Black, Flake8, Mypy, Pytest).
- `make build` — Сборка дистрибутивов (Wheel и SDist) через модуль `build`.
- `make publish` — Безопасная загрузка в TestPyPI с помощью `twine` через API-токен.
