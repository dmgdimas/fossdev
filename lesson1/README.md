Автоматизация управления проектом

Данный проект демонстрирует использование `Make` как единой точки входа для управления жизненным циклом Python-приложения.

Как запустить:
1. Клонируйте репозиторий.
2. Перейдите в папку проекта: `cd automation_project`.
3. Выполните установку: `make install`.
4. Запустите все проверки одной командой: `make check`.

Подробное описание архитектурных решений читайте в [отчете по проекту](./homework1/README.md).
